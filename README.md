# Charmday

모바일 특화 `사주 + 오행 + 상품 추천` 미니 서비스 MVP입니다.
이번 버전은 React 없이 정적 SPA(멀티-뷰)로 구성해 빠르게 테스트/배포할 수 있게 만들었습니다.

## 반영된 UX 요구사항

- 입력 방식: 셀렉트 제거, `연/월/일/시/분` 모두 타이핑 입력
- 미상 옵션:
  - `정확한 시간을 몰라요` -> 시/분 스킵
- 화면 분리:
  - 시작 페이지
  - 로딩 페이지 (`오늘의 부적템 찾는 중` + 인터랙티브 애니메이션 + 5초 카운트)
  - 결과 페이지 (카드형 상세 분석)
- 결과 강화:
  - 오행 밸런스 바
  - 우세/보강 기운, 음양 흐름, 럭키 무드
  - 메인 추천 + 추가 추천 리스트
- 폰트: `Pretendard` 사용
- 글로벌 전환: `한국어 / English / 日本語 / Español` 지원

## 실행

```bash
cd charmday
python3 -m http.server 5173
```

브라우저: `http://localhost:5173`

## 쿠팡 파트너스 설정

`app.js` 상단 `COUPANG_PARTNER_BASE` 값으로 파트너스 래핑 URL을 설정하세요.

- `""` (기본): 일반 쿠팡 검색 링크
- 값 지정: 쿠팡 검색 URL을 인코딩해서 파트너스 링크로 연결

## 주요 파일

- `index.html`: 시작/로딩/결과 3단계 UI
- `styles.css`: 모바일 퍼스트 스타일 + 로딩 애니메이션
- `app.js`: 다국어, 입력 검증, 분석 로직, 추천/링크 매핑
